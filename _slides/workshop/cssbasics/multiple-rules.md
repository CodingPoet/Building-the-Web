## More CSS Rules

CSS rules are just written one after another:

	.page-header {
		background-color: blue;
        color: white;
        font-size: 50px;
        padding: 30px;
	}

	.intro-section {
		background: orange;
        color: black;
        font-size: 14px;
        padding: 30px;
	}
